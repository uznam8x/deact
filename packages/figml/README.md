# Figml

Figma to html

```bash
npx figml -c <config> -f <figon file> -o <outout dir>
```

## Configuration

| key       | value            |
| --------- | ---------------- |
| resources | component assets |

```json
{
  "resources": {
    "Card": {
      "script": [],
      "stylesheet": []
    }
  }
}
```
